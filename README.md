# Test Front end

 ## Requerimientos
  * nodeJS versión v13.12.0 o superior 
  * yarn versión 1.22.4 o superior
  * git 

  ## Pasos para Configurar
   * create base de datos con script.sql

  ``` sh 
    * git clone git@github.com:AdrianMendez1199/ticket.git
    
    * cd ticket
    
    * yarn install    
    
    * yarn build 

    * yarn global add serve

  ```

  # Iniciar App
  * serve -s build
